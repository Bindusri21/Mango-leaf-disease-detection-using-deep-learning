# 🥭 Mango Leaf Disease Detection Using Deep Learning

## 📌 Project Overview

Mango Leaf Disease Detection Using Deep Learning is an AI-powered agricultural healthcare application designed to identify and classify mango leaf diseases using deep learning and computer vision techniques.

The system helps farmers and agricultural experts detect plant diseases at an early stage using image-based analysis, enabling faster treatment and improved crop productivity.

The project uses Convolutional Neural Networks (CNN) for image classification and provides an interactive web interface for disease prediction.

# ▶️ How to Run the Project

## 1. Clone Repository

```bash
git clone https://github.com/Bindusri21/Mango-leaf-disease-detection-using-deep-learning.git
```

---

## 2. Navigate to Project Folder

```bash
cd Mango-leaf-disease-detection-using-deep-learning
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Train the Model

```bash
python train.py
```

---

## 5. Run Streamlit Application

```bash
python3 -m streamlit run app.py
```
